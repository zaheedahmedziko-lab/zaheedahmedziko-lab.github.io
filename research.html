Research Interests

My research interests cover cybersecurity and secure computing across AI-enabled systems, networks, IoT, cloud computing, healthcare technologies, and future communication systems.

Cybersecurity AI Security AI Agent Security Machine Learning Security Secure Embedded Systems Network Security Software-Defined Networking IoT Security Cloud Computing Cloud Security Green Computing Healthcare Technologies Digital Forensics Quantum Computing Grid-Level Energy Storage Systems

Cybersecurity and Networked Systems

I am interested in the security of modern networks, including software-defined networking, control-plane security, network attack analysis, secure communication systems, and privacy-aware network defense.

AI and Secure Intelligent Systems

I am interested in the security of AI-enabled systems, including AI agents, tool-using systems, adversarial behavior, indirect prompt injection, secure AI workflows, and trustworthy computing.

IoT, Cloud, and Healthcare

My work also connects IoT, cloud computing, edge intelligence, and healthcare technologies, with attention to scalable architectures, real-time monitoring, and secure system design.

Emerging Technologies

I also maintain broader interests in quantum computing, green computing, embedded systems security, future internet technologies, and grid-level energy storage systems.
